# Instalar

1. Instalar NodeJS
2. Criar App React
    `npm create vite <NomeProjeto>`
    `npm isntall`
3. `npm run dev` ~dentro do projeto
4. Instalar PrimeReact
    `npm install primereact`
    `npm install primeicons`
